# Gitのオフィシャルサイト  
https://git-scm.com/  

# Javaのインストール状況の確認  
java -version  

# Gitをインストール後に行う設定  
$ git config --global user.name "Mei Sei"  
$ git config --global user.email "hoge@email.com"  
$ git config --global core.quotepath false  
$ git config --global gui.encoding utf-8  


# 単体のJavaファイルの実行方法  
## 1. コンパイル  
javac ファイル名.java
※上のコマンドを実行したらファイル名.classが生成されます。  
## 2. 実行方法  
java ファイル名  
上記で実行できます。
